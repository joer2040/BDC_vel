# Velocity / KPMG Color Usage Guidelines
> Apply these guidelines to ALL visual outputs: dashboards, charts, infographics, HTML/CSS, presentations, and any AI-generated visual code.

---

## Rules (enforce always)
- **Prioritize blues** as the dominant palette — KPMG Blue `#00338D` is the primary brand anchor
- **Do NOT use all colors at once** — select a purposeful subset per visual
- **Mix light, mid, and dark tones** within the same data set for contrast and hierarchy
- **No gradients** on backgrounds of infographics or charts
- **Neutrals are for charts/infographics only** — not for general UI backgrounds or text

---

## 1. Primary Colors
Use for headers, key UI elements, backgrounds, and primary data series.

| Name           | HEX       | RGB              | Use                              |
|----------------|-----------|------------------|----------------------------------|
| KPMG Blue      | `#00338D` | 0, 51, 141       | Primary brand color, headers, CTA buttons |
| Cobalt Blue    | `#1E49E2` | 30, 73, 226      | Active states, highlights, links |
| Spectrum Blue  | `#0C233C` | 12, 35, 60       | Dark backgrounds, shells, nav bars |
| Light Blue     | `#ACEAFF` | 172, 234, 255    | Light fills, hover states, backgrounds |
| Pacific Blue   | `#00B8F5` | 0, 184, 245      | Accent lines, chart series, KPI values |
| Purple         | `#7213EA` | 114, 19, 234     | Secondary accent, badges, callouts |
| Pink           | `#FD349C` | 253, 52, 156     | Alerts, highlights, tertiary accent |

## 2. Accent Colors — Infographics & Charts

| Name           | HEX       | RGB              | Use                              |
|----------------|-----------|------------------|----------------------------------|
| Blue           | `#76D2FF` | 118, 210, 255    | Light data series, area fills    |
| Dark Purple    | `#510DBC` | 81, 13, 188      | Contrast series against blues    |
| Light Purple   | `#B497FF` | 180, 151, 255    | Secondary data series            |
| Dark Pink      | `#AB0D82` | 171, 13, 130     | Risk/alert data series           |
| Light Pink     | `#FFA3DA` | 255, 163, 218    | Low-priority or tertiary series  |

| Green          | `#00C0AE` | 0, 192, 174      | Favorable trends, approvals      |
| Light Green    | `#63EBDA` | 99, 235, 218     | Light positive fills, backgrounds|

## 3. Neutrals — Infographics & Charts Only
Use for gridlines, labels, borders, and secondary text within charts. Not for general UI.

| Name    | HEX       | RGB              | Use                              |
|---------|-----------|------------------|----------------------------------|
| Grey 1  | `#333333` | 51, 51, 51       | Chart body text, axis labels     |
| Grey 2  | `#666666` | 102, 102, 102    | Secondary labels, annotations    |
| Grey 3  | `#989898` | 152, 152, 152    | Gridlines, dividers              |
| Grey 4  | `#B2B2B2` | 178, 178, 178    | Disabled states, placeholders    |
| Grey 5  | `#E5E5E5` | 229, 229, 229    | Background fills, zebra rows     |
| White   | `#FFFFFF` | 255, 255, 255    | Text on dark backgrounds, cards  |

## Recommended Chart Color Order
When assigning multiple series in a chart, follow this priority order:

1. `#00338D` — KPMG Blue
2. `#1E49E2` — Cobalt Blue
3. `#76D2FF` — Blue (accent)
4. `#00B8F5` — Pacific Blue
5. `#ACEAFF` — Light Blue
6. `#510DBC` — Dark Purple
7. `#B497FF` — Light Purple
8. `#7213EA` — Purple
9. `#098E7E` — Dark Green
10. `#00C0AE` — Green
11. `#63EBDA` — Light Green
12. `#AB0D82` — Dark Pink
13. `#FD349C` — Pink
14. `#FFA3DA` — Light Pink

## Quick Reference — Common Pairings

| Scenario                        | Recommended Colors                              |
|---------------------------------|-------------------------------------------------|
| Dark background / shell         | `#0C233C` bg + `#ACEAFF` or `#FFFFFF` text      |
| Primary CTA button              | `#00338D` fill + `#FFFFFF` text                 |
| Positive / success indicator    | `#098E7E` or `#00C0AE`                          |
| Alert / risk indicator          | `#AB0D82` or `#FD349C`                          |
| Chart multi-series (3 series)   | `#00338D`, `#00B8F5`, `#76D2FF`                 |
| Chart multi-series (5 series)   | `#00338D`, `#1E49E2`, `#76D2FF`, `#510DBC`, `#00C0AE` |
| Neutral table rows (zebra)      | `#FFFFFF` + `#E5E5E5`                           |
| Confidence / progress bars      | `#00338D` (high) → `#00B8F5` (mid) → `#B497FF` (low) |
